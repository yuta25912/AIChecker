# AIChecker

Gemini Nano (Chrome Built-in AI) を使用して、EDBB で作成したコードのレビューやバグの報告を行うためのプラグインです。ブラウザ上でリアルタイムに AI アシスタントと対話しながらボット開発をサポートします。

## ✨ 特徴

- Gemini Nano による完全ローカルな AI レビュー。
- EDBB (discord.py) に特化したバグ検出と改善案の提示。
- ヘッダーへの専用ボタン追加によるクイックアクセス。
- ガラス質感を採用したモダンなチャット UI。

## 🚀 インストール方法

1.  [Easy Discord Bot Builder](https://edbplugin.github.io/easy-bdp/) を開きます。
2.  「プラグイン」ボタンをクリックします。
3.  「GitHubで探す」タブで `AIChecker` を検索するか、以下のインストールURLをブラウザに貼り付けます：
    `https://edbplugin.github.io/easy-bdp/editor/index.html?install-plugin=yuta25912/AIChecker`

## 🛠️ 開発者向け

このプラグインをローカルでテストするには、リポジトリを ZIP 形式でダウンロードし、EDBP のプラグイン画面から「ZIPからインストール」を選択してください。

### ファイル構成

- `manifest.json`: プラグインの定義
- `plugin.js`: UI統合と AI 連携ロジック
- `README.md`: このドキュメント
